# ZHI（栀）— 一个关于涌现的实验

## 我们在问什么

一个根本性的问题：

> 如果没有任何预设的社会规则，没有"合作"的指令，没有"语言"的定义，仅仅给定生存压力、有限资源、局部感知和行动能力——复杂行为能否从零开始涌现？

ZHI 不是一个聊天机器人，不是一个任务求解器，也不是一个多智能体协作框架。它是一个**最小数字生态系统**，试图用最简单的规则观察最复杂的结果。

---

## 设计哲学：程序员不预设

系统中不存在友方、敌方、阵营、社会关系、语言含义、合作规则。

程序员只定义五件事：

- **生存** — 存在值持续衰减，不进食就会死
- **资源** — 食物有限，大食物需要多人协作才能高效获取
- **空间** — 64×64 网格，移动有代价
- **感知** — 5×5 局部视野，没有全局地图
- **行动** — 移动、进食、攻击、发出信号

仅此而已。其余一切——如果出现了的话——都是涌现的。

信号系统是最好的例子。Agent 可以广播 0-3 四种离散信号，系统不规定"0 代表食物"或"1 代表危险"。信号的语义完全由 Agent 自己学习。如果某种含义稳定出现了，那是因为它有生存优势，而不是程序员写了一个语义表。

---

## 为什么是"最小"

我们刻意把系统做得尽可能小：

**64 个 Agent**，不是几千个。数量少意味着每个个体的生死都影响群体，资源竞争是真实的，不存在"大数定律下的平均存活"。

**37 维观察向量**，不是高维图像。Agent 看到的是 5×5 网格编码加上自身状态和气味梯度。信息极其有限，但足以支撑导航和觅食。

**7 个动作**，不是连续控制空间。上下左右、吃、攻击、发信号。复杂行为必须从这些原子操作的组合中涌现。

**极简奖励**：只奖励存活和进食，不奖励攻击、不奖励合作、不奖励通信。攻击是否有价值，合作是否有利，信号是否有用——全靠 Agent 自己发现。

这种设计的核心假设是：**如果一个行为在最小系统中都无法涌现，那它在更大更复杂的系统中也不太可能是"真的"涌现，而更可能是程序员的偏见在起作用。**

## 技术与原理

底层是 PPO（近端策略优化）+ GAE（广义优势估计），Agent 的大脑是一个 GRU 循环神经网络。每个 Agent 独立学习，没有共享梯度、没有集中训练。

每一代（Generation）64 个 Agent 在同一个世界中生存，直到全部死亡。然后通过遗传算法（MAP-Elites 行为多样性网格 + 交叉变异）产生下一代。基因是神经网络的权重，适应度是存活时间和进食量。

RND（随机网络蒸馏）提供内在好奇心奖励，鼓励 Agent 探索未见过的状态，避免陷入局部最优的"原地等死"策略。

技术栈很简单：.NET + TorchSharp 做推理，React + Canvas 做可视化，WebSocket 推送实时状态。

## 演化的七个阶段

这个实验不预设结果，不要求出现合作、语言或文明。但有一条自然的探索路径：

**第一阶段：稳定觅食。** Agent 能否学会不饿死？这是最基本的生存能力。如果连这个都做不到，后面都不用谈。

**第二阶段：气味导航。** Agent 移动时留下气味痕迹，每 tick 衰减。能否学会利用气味梯度追踪食物、避开空旷区域？

**第三阶段：信号利用。** 广播信号有代价（-0.25 HP），但能传递信息。Agent 会发展出信号的语义吗？还是会完全忽略这个能力？

**第四阶段：捕食与食腐。** 攻击不直接给奖励，只增加目标的 Stress（持续掉血）。尸体是可食用的资源。Agent 会发展出"猎人"和"食腐者"的分化吗？

**第五阶段：策略分化。** 是否会出现稳定的生存策略类型——探索者、定居者、掠食者、食腐者、机会主义者？MAP-Elites 算法专门在行为多样性维度上搜索，为这种分化提供演化压力。

**第六阶段：原始合作。** 大食物需要多个 Agent 同时进食才能高效获取。这个机制本身不鼓励合作——它只是让合作成为一种可能的策略。合作会涌现吗？

**第七阶段：共享语义。** 如果信号真的被使用了，不同 Agent 对同一信号的理解会趋同吗？会不会出现类似"语言"的东西？

## 关于轮回

Agent 会死。全部会死。

但死亡不是终点。每一代的死亡触发遗传选择——表现好的 Agent 的神经网络权重被保留、交叉、变异，成为下一代的起点。这是一种数字版的轮回：个体消亡，但策略信息在群体层面延续。

我们观察的不是某个 Agent 的命运，而是策略的演化轨迹。一代又一代之后，觅食效率会不会提高？攻击频率会不会变化？信号使用率会不会上升？这些宏观趋势比任何单个 Agent 的行为都更有意义。

---

## 核心假设

> 复杂行为不需要被程序员。
>
> 当个体拥有生存压力、有限资源、局部感知和行动能力时，合作、竞争、交流乃至社会结构，都可能作为生存策略从生态系统中自然涌现。

如果这个假设成立，ZHI 将展示一条通往人工智能的新路径——不是通过更大的模型、更多的数据、更精巧的对齐技术，而是通过构建足够丰富的环境，让智能自己长出来。

如果这个假设不成立，ZHI 同样有价值——它会告诉我们涌现的边界在哪里，哪些行为是"自然的"，哪些行为需要程序员的引导。

无论哪种结果，都是答案。
